الدردشة المجتمعية

كل الكشّافة يشاركون في مجتمع واحد بعد كل مباراة:

  • اكتب منشورًا قصيرًا (حتى ٢٨٠ حرفًا) عن أي مباراة أو رأيك في النتائج.
  • اربط منشورك بمباراة لربط النقاش بها.
  • تفاعل مع منشورات الآخرين برمز تعبيري واحد.
  • علّق ضمن المنشور للنقاش الموسّع.
